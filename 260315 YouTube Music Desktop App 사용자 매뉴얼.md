# YouTube Music Desktop App 사용자 매뉴얼

> 광고 없이, 브라우저 없이, 음악만 깔끔하게.
> YouTube를 데스크톱 음악 플레이어로 바꿔주는 앱입니다.

---

## 이런 분들을 위한 앱이에요

- YouTube에서 음악을 자주 듣는데, 브라우저 탭을 따로 열어두기 귀찮은 분
- YouTube Premium 없이도 **백그라운드 재생**을 하고 싶은 분
- 작업하면서 화면 구석에 **작은 미니플레이어**를 띄워두고 싶은 분
- **가사**를 보면서 음악을 듣고 싶은 분
- 나만의 **플레이리스트**를 만들어서 연속 재생하고 싶은 분

---

## 설치하기

1. `YouTube-Music-Setup-x.x.x.exe` 파일을 실행합니다
2. 설치가 자동으로 진행됩니다 (원클릭 설치)
3. 바탕화면 또는 시작 메뉴에서 **YouTube Music**을 실행합니다

> 이미 이전 버전이 설치되어 있다면, 그냥 새 버전을 설치하면 자동으로 업그레이드됩니다. 기존 설정, 즐겨찾기, 재생 기록은 모두 유지됩니다.

---

## 첫 실행

앱을 처음 열면 YouTube 화면이 나타납니다. 평소 브라우저에서 YouTube를 사용하듯이 로그인하고 음악을 재생하세요.

앱 상단에는 **커스텀 타이틀바**가 있어요:

| 버튼 | 기능 |
|------|------|
| ← | 이전 페이지 |
| → | 다음 페이지 |
| 🏠 | YouTube 홈으로 이동 |

## 핵심 기능 소개

### 1. 백그라운드 재생

YouTube는 원래 다른 탭으로 이동하거나 브라우저를 최소화하면 재생이 멈춥니다.
이 앱에서는 **창을 최소화해도, 다른 작업을 해도 음악이 계속 재생**됩니다.
Premium 없이도요.

---

### 2. 미니 플레이어

메인 창을 최소화하면 화면 하단에 **작은 미니 플레이어**가 자동으로 나타납니다.

#### 컴팩트 모드 (기본)

작고 깔끔한 위젯 형태입니다. 화면 모서리에 붙어서 작업을 방해하지 않아요.

표시되는 정보:
- 앨범아트 (썸네일)
- 곡 제목 (길면 자동으로 흘러감)
- 아티스트
- 재생 진행률 바

조작 버튼:
- ◀ 이전곡
- ▶ 재생/일시정지
- ▶▶ 다음곡
- ✕ 닫기

#### 확장 모드

더 큰 화면으로 더 많은 정보를 볼 수 있습니다.

추가로 보이는 것:
- 큰 앨범아트
- 현재 시간 / 전체 시간
- PIP 버튼, 가사 버튼
- 최근 재생 목록

#### 모드 전환 방법

- **미들클릭** (마우스 휠 클릭) → 컴팩트 ↔ 확장 전환
- **트레이 메뉴** → 설정 → 미니플레이어 크기

#### 미니 플레이어 마우스 조작

| 조작 | 동작 |
|------|------|
| 더블클릭 | 가사 팝업 열기/닫기 |
| 미들클릭 (휠 클릭) | 컴팩트/확장 전환 |
| 우클릭 | 컨텍스트 메뉴 (아래 설명) |
| 드래그 | 미니플레이어 위치 이동 |
| Ctrl + 마우스 휠 | 투명도 조절 |

#### 우클릭 컨텍스트 메뉴

미니플레이어를 우클릭하면:
- **곡 링크 복사** — 현재 곡의 YouTube URL 복사
- **곡 제목 복사** — "아티스트 - 제목" 형식으로 복사
- **가사 팝업** — 가사 창 열기/닫기
- **설정** — 투명도, 볼륨, 속도 등 설정
- **미니플레이어 닫기**

---

### 3. 가사 보기

음악을 들으면서 가사를 함께 볼 수 있어요. LRCLIB이라는 무료 가사 데이터베이스를 사용합니다.

#### 가사 여는 방법

- 단축키 `Ctrl + L` (메인 창 사이드 패널)
- 미니플레이어 **더블클릭** (가사 팝업)
- 트레이 메뉴 → 가사

#### 가사 팝업 (5개 탭)

미니플레이어 위에 뜨는 가사 팝업에는 5개의 탭이 있습니다:

**가사 탭**
- 현재 재생 위치에 맞춰 가사가 자동으로 하이라이트됩니다
- 가사를 클릭하면 해당 시점으로 점프합니다
- 가사 타이밍이 안 맞으면 하단의 `-0.5` / `+0.5` 버튼으로 조절하세요
- 조절한 싱크는 곡별로 자동 저장됩니다 (다음에 같은 곡을 틀면 자동 적용)

**재생목록 탭**
- 내가 만든 커스텀 플레이리스트 (큐) 목록
- 현재 재생 중인 곡이 하이라이트 표시됩니다
- 곡을 클릭하면 바로 재생
- 개별 삭제, 전체 삭제 가능

**즐겨찾기 탭**
- 즐겨찾기에 추가한 곡 목록
- 클릭하면 바로 재생

**히스토리 탭**
- 최근 재생한 곡 50개 기록
- 클릭하면 바로 재생

**검색 탭**
- YouTube 영상/플레이리스트를 직접 검색
- 검색 결과에서 바로 재생하거나, 내 재생목록에 추가 가능

---

### 4. 나만의 플레이리스트 (커스텀 큐)

YouTube 플레이리스트와 별개로, 앱 안에서 **나만의 재생 목록**을 만들 수 있습니다.

#### 곡 추가하기

**방법 1: 검색으로 추가**
1. 가사 팝업 → 검색 탭
2. 듣고 싶은 곡 검색
3. 검색 결과 옆의 **+** 버튼 클릭

**방법 2: 플레이리스트 통째로 추가**
1. 가사 팝업 → 검색 탭
2. 검색 모드를 "플레이리스트"로 변경
3. 플레이리스트 클릭 → 상세 보기
4. **"전체 추가"** 버튼 → 전체 곡이 한번에 추가됩니다
5. 또는 개별 곡 옆의 **+** 버튼으로 원하는 곡만 추가

#### 큐 동작 방식

- 큐에 곡이 있으면, 현재 곡이 끝나면 **큐의 다음 곡이 자동 재생**됩니다
- 다음곡/이전곡 버튼도 **큐 순서**대로 동작합니다
- 큐의 마지막 곡이 끝나면 첫 곡부터 다시 순환합니다
- 큐 데이터는 로컬에 저장되어 **앱을 재시작해도 유지**됩니다

#### 큐 관리

가사 팝업 → 재생목록 탭에서:
- 곡 클릭 → 해당 곡 재생
- 곡 옆의 X 버튼 → 개별 삭제
- 전체 삭제 버튼 → 큐 초기화

---

### 5. 광고 자동 스킵

광고가 재생되면 미니플레이어에 **"AD"** 표시가 나타나서 현재 광고 중임을 알 수 있어요. 화면에는 보이지 않지만, 내부적으로 "광고 건너뛰기" 버튼이 나타나는 순간 **자동으로 눌러서** 바로 음악으로 돌아갑니다. 스킵 버튼이 없는 짧은 광고도 자동으로 빠르게 넘겨줍니다.

> **Tip:** 유니콘(Unicorn) 같은 광고 차단 앱을 사용하고 계신다면, 이 앱(YouTube Music)을 광고 차단 목록에 추가해보세요. 광고 대기 없이 바로바로 음악을 들을 수 있습니다.

---

### 6. 시스템 트레이

앱을 닫아도(X 버튼) 완전히 종료되지 않고, **시스템 트레이**(작업표시줄 오른쪽 아이콘 영역)에 남아있습니다.
음악은 계속 재생됩니다.

#### 트레이 아이콘 조작

| 조작 | 동작 |
|------|------|
| 더블클릭 | 메인 창 다시 열기 |
| 우클릭 | 트레이 메뉴 열기 |

#### 트레이 메뉴에서 할 수 있는 것

- 즐겨찾기에 현재 곡 추가/삭제
- 최근 재생 목록에서 곡 선택
- 설정 변경 (투명도, 볼륨, 위젯 위치 등)
- 설정 → 실험적 기능 → 메인창 열기 / MV→음원 자동 전환

---

### 7. 즐겨찾기

마음에 드는 곡을 즐겨찾기에 저장해두면 언제든 빠르게 재생할 수 있어요.

**추가하기:**
- 트레이 메뉴 → "즐겨찾기 추가"

**재생하기:**
- 트레이 메뉴 → 즐겨찾기 → 곡 선택
- 가사 팝업 → 즐겨찾기 탭

---

### 8. MV → 음원 자동 전환 (실험적 기능)

뮤직비디오(MV)를 재생하면, 자동으로 같은 곡의 **Official Audio** 버전을 찾아서 전환해줍니다. MV에는 인트로, 중간 인서트, 아웃트로 같은 음원에 없는 내용이 포함되어 있어서 가사 싱크가 안 맞는 경우가 많아 만들어 놓은 기능입니다.

다만, 아직 **실험적 기능**입니다. 음원을 정확히 찾아주기도 하지만, 가끔 같은 곡의 다른 버전(리믹스, 라이브 등)을 찾아오는 경우도 있어요. 참고하고 사용해주세요.

> 기본적으로 꺼져 있습니다. 트레이 메뉴 → 설정 → "MV → 음원 자동 전환"을 체크하면 켜집니다.

## 설정 안내

트레이 메뉴 → 설정, 또는 미니플레이어 우클릭 → 설정에서 변경할 수 있습니다.

| 설정 | 설명 | 기본값 |
|------|------|--------|
| 제목 흐르기 | 긴 제목이 자동으로 흐르는 효과 켜기/끄기 | 켜짐 |
| 흐르기 속도 | 느리게 / 보통 / 빠르게 / 매우 빠르게 | 보통 |
| 투명도 | 미니플레이어 투명도 (100% ~ 15%) | 100% |
| 볼륨 | 100% / 75% / 50% / 25% / 무음 | 100% |
| 미니플레이어 크기 | 컴팩트 / 확장 | 확장 |
| 위젯 위치 | 좌상단 / 우상단 / 좌하단 / 우하단 | 좌하단 |

#### 실험적 기능

| 설정 | 설명 | 기본값 |
|------|------|--------|
| MV → 음원 자동 전환 | 뮤직비디오를 음원으로 자동 전환 (정확도가 완벽하지 않음) | 꺼짐 |
| 메인창 열기 | YouTube 메인 브라우저 창을 다시 엽니다 | — |

## 데이터 저장 위치

모든 설정과 데이터는 아래 폴더에 자동 저장됩니다:

```
C:\Users\{사용자}\AppData\Roaming\youtube-music\
```

| 파일 | 내용 |
|------|------|
| `config.json` | 앱 설정 (창 크기, 투명도, 위치 등) |
| `history.json` | 최근 재생 기록 (최대 50곡) |
| `favorites.json` | 즐겨찾기 목록 |
| `queue.json` | 커스텀 플레이리스트 |
| `lyrics-sync.json` | 곡별 가사 싱크 보정값 |

> 앱을 업그레이드해도 이 데이터는 그대로 유지됩니다.

## 자주 묻는 질문

**Q. 앱을 닫았는데 음악이 계속 나와요.**
A. 정상입니다! X 버튼을 누르면 트레이로 최소화됩니다. 완전히 종료하려면 트레이 아이콘 우클릭 → "종료"를 선택하세요.

**Q. 가사가 안 나와요.**
A. 모든 곡에 가사가 있는 것은 아닙니다. LRCLIB 데이터베이스에 해당 곡의 가사가 등록되어 있어야 표시됩니다. 한국 노래는 특히 가사가 없는 경우가 많을 수 있어요.

**Q. 가사 타이밍이 안 맞아요.**
A. 가사 팝업 하단의 `-0.5` / `+0.5` 버튼으로 조절하세요. 조절한 값은 곡별로 자동 저장됩니다.

**Q. 이전 버전 위에 새 버전을 설치해도 되나요?**
A. 네, 그냥 새 버전 설치 파일을 실행하면 자동으로 업그레이드됩니다. 기존 설정, 즐겨찾기, 재생기록, 플레이리스트 모두 유지됩니다.

**Q. 미니플레이어가 안 보여요.**
A. 트레이 아이콘 우클릭 → "미니 플레이어"를 클릭하거나, `Ctrl + M`을 눌러보세요. 또는 메인 창을 최소화하면 자동으로 나타납니다.

**Q. 설치 시 SmartScreen 경고가 떠요.**
A. 코드사이닝이 되어있지 않아서 뜨는 정상적인 경고입니다. "추가 정보" → "실행"을 클릭하면 설치할 수 있습니다.

## 배포 및 사용 안내

이 앱은 개인적으로 만들어서 **지인들에게만 공유하는 비공개 앱**입니다.

- 전달받은 분만 사용해주세요
- **다른 사람에게 재배포하거나 공유하지 말아주세요**
- 문제가 생기면 개발자(rockuen)에게 직접 연락해주세요
- 업데이트가 있으면 개별적으로 공유드리겠습니다

이 앱은 **개인정보를 일절 수집하지 않습니다.** 로그인 정보, 재생 기록, 검색 내역 등 어떤 사용자 데이터도 외부 서버로 전송하거나 별도로 저장하지 않습니다. 모든 데이터(설정, 즐겨찾기, 재생기록, 플레이리스트)는 본인 PC의 로컬 폴더에만 저장됩니다.
